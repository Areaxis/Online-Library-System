# Online Library System (React & Vite)

A simple and interactive online library management application built primarily using React and Redux Toolkit.  
This project allows users to browse, search, view, and add books dynamically with a clean and responsive user interface.

## Libraries Used

*React (Vite)
*Redux Toolkit
*React Router DOM

## Features

*Browse books by category  
*Search books by title or author (can detect wrapped text)  
*View detailed information about each book  
*Add new books using a form  
*Dynamic routing using React Router  
*Global state management using Redux Toolkit  
*Responsive design with animations  
*404 page for invalid routes

## How to install

Use "npm install" in order to installed the required dependencies and run "npm run dev" in order to begin hosting in your localhost.
